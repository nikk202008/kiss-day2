# Kiss Day Webpage

Cute website using your GIF:

- `kisssses.gif`

Now it’s **two pages**:

- `index.html` (permission page)
- `yes.html` (cute “YES” page)

## How to open

Just double-click `index.html`.

If your browser blocks some features (like clipboard), run a tiny local server:

### Option A (Python)

```bash
python -m http.server 5173
```

Then open `http://localhost:5173`.

### Option B (Node)

```bash
npx serve .
```
